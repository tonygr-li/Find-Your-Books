Bienvenue sur notre site web! Nous sommes ici pour vous aider à naviguer à travers le site. Si jamais nous ne répondons pas à vos question sur cette page, n'hésitez pas à nous écrire à trouveteslivres@gmail.com, et il nous fera plaisir de vous aider!

Vous devez vous inscrire. Ensuite, vous recevrez un courriel de confirmation afin de valider votre identité. Toutefois, il se peut que ce courriel se trouve dans vos "spam", alors assurez-vous de vérifier toutes vos boites de réception. Si après 10 minute vous n'avez toujours pas reçu le courriel de confirmation, vous pouvez réessayer d'activer votre comptre en vous rendant sur la page de connexion) et vous pouvez réessayer de vous inscire.

Vous devez vous inscire pour publier une annonce (Vente ou Recherche d'un livre), ou pour signaler un utilisateur. Vous n'avez pas besoin d'un compte pour acheter un livre ou pour voir les annonce.

Le site web utilise contains la recherche rapide. Ceci signifie que le moteur de recherche vous montre toutes les annonces qui contiennent les éléments que vous avez entré. Par exemble, si vous recherchez l'ISBN «123», le moteur de recherche truvera toute les occurences pour «123» dans notre base de données. Ceci signifie que les résultats inclueraient «1234567891011», «2341233447672», et «3212394583582», mais pas «1, 2, 3» ou «1-2-3». Voici pourquoi chercher l'ISBN du livre (versus le nom de l'auteur ou le titre du livre) est le moyen de recherche le plus efficace pour vendre ou acheter un livre.
